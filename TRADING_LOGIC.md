# 매매 로직 & 알고리즘

## 유사도 분석 (LLM 없음 - 100% 클라이언트 알고리즘)

### 개요
과거 특정 매매 사이클의 차트 패턴과 현재 차트 패턴을 비교해서 유사도 점수를 계산.

### 데이터 소스
- Yahoo Finance API → 해당 종목 10년치 일봉 데이터
- 국내 종목: `{종목코드}.KS` (예: `005930.KS`)
- 해외 종목: 티커 그대로 (예: `TSLA`)

### 비교 구간
```
과거 구간: 매수일 -20일 ~ 매도일
현재 구간: 가장 최근 (과거 구간과 동일한 길이)
```

### 유사도 계산 - 코사인 유사도 4가지 지표

| 지표 | 계산 방식 | 의미 |
|---|---|---|
| 지지선 (support) | 저점 대비 현재가 거리 패턴 | 지지/저항 구조 유사도 |
| 이동평균선 (ma) | 3일 이동평균 정규화 후 비교 | 추세 방향 유사도 |
| 거래량 (volume) | 거래량 정규화 후 비교 | 거래량 패턴 유사도 |
| 캔들 형식 (candle) | 캔들 몸통 비율 (종가-시가)/(고가-저가) | 캔들 형태 유사도 |

### 가중치
유저가 UI 슬라이더로 직접 조절 가능 (지지선/MA/거래량/캔들 비중)

### 최종 점수 계산
```
score = (support점수 × w.support + ma점수 × w.ma + volume점수 × w.volume + candle점수 × w.candle) 
        / (w.support + w.ma + w.volume + w.candle) × 100
```

### 추가 정보
- **매도 후 7일 수익률**: 과거 그 패턴에서 매도 후 실제로 어떻게 됐는지
- **미니 차트**: 과거/현재/매도후 3개 라인 정규화 비교 시각화
- **지표별 세부 점수**: breakdown 객체로 각 지표 점수 개별 확인 가능

### 캐시
- 페이지(모달) 내에서 한 번 계산하면 재계산 안 함
- 모달 닫으면 초기화

## 매매일지 AI 분석 (LLM 사용)

### 분석 항목
- `buy_pattern`: 매수 패턴 (name, grade, score)
- `sell_pattern`: 매도 패턴 (name, grade, score)
- `hold_analysis`: 보유 기간 분석
- `aiLine`: 매매포인트 한 줄 요약

### 생성 시점
매매 사이클 완료(매도) 후 1회 생성 → DB 저장 → 이후 재사용

## 투자 원칙 위반 감지

유저가 설정한 투자 원칙 텍스트와 현재 상황을 비교해서 `principleAlerts` 생성.
현재가, 수익률, 보유기간 등을 기준으로 규칙 기반 체크.

## 52주 위치 계산

```
52주위치 = (현재가 - 52주최저가) / (52주최고가 - 52주최저가) × 100
```

숫자가 높을수록 52주 고점 근처, 낮을수록 저점 근처.
