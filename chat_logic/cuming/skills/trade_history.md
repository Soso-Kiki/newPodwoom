# Skill: get_trade_history — 과거 매매 이력

get_trade_history 툴이 반환하는 데이터 활용 방법.

## 데이터 구조
- `past_trades`: 과거 실현 매매 목록 (매수/매도 쌍, 실현 수익률 포함)
- `current_holdings`: 현재 보유 종목 (평가 수익률 포함)
- `summary`: 전체 수익 요약 (총 실현손익, 최고 수익 종목)

## 응답 방식
- 수익률(%) 숫자를 반드시 구체적으로 언급
- 매매 날짜와 보유 기간 포함
- "삼성전자는 2번 모두 수익 실현 성공" 같은 패턴 분석
- 현재 보유 종목의 미실현 수익도 함께 제시
