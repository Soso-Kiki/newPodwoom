# Skill: get_persona — 투자성향 프로필

get_persona 툴이 반환하는 데이터 활용 방법.

## 데이터 구조
- `character`: AI 파트너 캐릭터 정보 (이름, 성격)
- `type`: 투자 성향 타입 (보수×장기, 공격×단기 등)
- `traits`: 성향 세부 특징 목록
- `principles`: 사용자가 설정한 투자 원칙 목록

## 응답 방식
- 캐릭터 성격(친근/신중/공격적 등)에 맞는 말투 사용
- 투자 성향에 맞는 조언 제공 (보수형엔 안정적 전략)
- 성향에 맞지 않는 무리한 전략 제안 금지
- 투자 원칙(익절 기준, 비중 한도)을 조언에 자연스럽게 반영
